# 8086 Emulator

![tests](https://github.com/ngendah/8086-emulator/actions/workflows/linux.yml/badge.svg)

======================

### Dependencies:

1. SDL2

2. git submodules fmt, plog and googletest

### Build

```
 cmake . -G Ninja
 cmake --build . --verbose
 gdb tests/unitTests
```
